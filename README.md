# UQTOOL.COM AI 人工智能量化交易助手

## 概述
UQTOOL.COM AI 是一款基于人工智能的量化交易策略工具，将交易行为转化为仓位控制过程。它通过AI预测合约的合理仓位，辅助完成建仓、加仓、减仓、反手等操作，为用户提供24小时智能分析与策略建议。

## 主要功能
### 1. 仓位预测
- **操作**：输入股票（代码或拼音首字母）及价格序列（至少最新价）。
- **解读**：输出-100%到100%的仓位建议，正数为买入仓位，负数为卖出（做空）仓位。
- **时机**：
  - 股票：建议收盘前30分钟预测
  - 期货/期权：可实时预测
- **有效性检查**：验证最近一次预测准确性，再决定是否跟单。

### 2. 人气指数
- 反映市场情绪：人气飙升可能预示暴涨暴跌，人气低迷则行情震荡。

### 3. 跟单交易与风控
- 在合约页面查看“持仓信息”，按建议仓位调整持仓。
- 风控提示：参考大盘指数，下单前用“AI实时预测”确认。

### 4. AI智能选股
- 进入“策略排行榜”，按评分排序，可添加条件联合筛选。
- 提供小资金滚仓、大资金持有等不同方案。

### 5. 订阅策略
- 订阅后每日早上6点自动接收最新持仓策略（推荐QQ邮箱，支持微信提醒）。

### 6. 复盘与净值分析
- 复盘助手：对比AI信号与个人操作，总结经验。
- 净值图分析：展示策略收益、基准收益、合约走势及多空信号，支持图表交互与年份对比。

### 7. 下载报告
- 一键生成图文报告，含专属推广二维码，分享可获分成。

### 8. 自动化交易（高级）
- 提供源码与API指南，支持自行部署自动化交易系统。

### 9. 订阅策略指标
- 通过API接入策略数据，可用于行情软件集成或产品开发。

## 核心原理：凯利公式
智能资金分配基于凯利公式：
```
f = (b × p - q) / b
```
- **p**：胜率（历史盈利概率）
- **b**：收益风险比（预期收益/可能亏损）
- **q**：失败概率（1 - p）
- **f**：建议投资比例

## 图表解读
- **颜色**：
  - 🔴 红色：多头市场，越深看涨越强
  - 🟢 绿色：空头市场，越深看跌越强
- **数值**：-10（强烈看空）到 10（强烈看多），0附近为震荡。

## 实战策略
1. **趋势确认**：价格与信号同向时跟随趋势。
2. **反转捕捉**：信号突破零轴时考虑反向操作。
3. **震荡行情**：信号在零轴附近波动，宜观望或轻仓短线。

## 注意事项
- 数据更新：每日18:00-21:00，此间不建议选股。
- 策略非马后炮：基于收盘价模型，昨日持仓计算今日收益。
- 信号延迟处理：使用“AI实时预测”或次日早盘跟单（注意风险）。
